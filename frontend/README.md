# Survey App Frontend

This is the frontend for the Survey App, built with **React 19**, **Vite**, **HeroUI**, and **Tailwind CSS**.

## Features
- **Modern UI**: Powered by HeroUI (formerly NextUI) with glassmorphism effects.
- **Dark Mode**: Automatic system-level detection and manual toggle.
- **Accessibility**: Built on React Aria primitives for WCAG compliance.
- **Responsive Design**: Mobile-friendly layouts and components.

## Tech Stack
- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Component Library**: [HeroUI](https://heroui.com/)
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run in development mode**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Key Directories
- `src/components`: Reusable HeroUI components and question types.
- `src/pages`: Application pages (Dashboard, Browse, Take Survey, etc.).
- `src/contexts`: React contexts for Auth and Theme management.
- `src/layouts`: Base layout components.
